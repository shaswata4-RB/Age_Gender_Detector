# 📌 Overview

Age & Gender Detector is a Computer Vision project that predicts the **approximate age** and **gender** of a person from an image or webcam feed using Deep Learning models and OpenCV.

The project demonstrates practical applications of Artificial Intelligence in facial analysis and real-time computer vision.

---

# ✨ Features

- 👤 Face Detection
- 🧠 Deep Learning-based Age Prediction
- 🚻 Gender Classification
- 📷 Webcam Support
- 🖼️ Image Detection
- ⚡ Fast Inference using OpenCV DNN
- 💻 Easy to Run

# 🧠 How It Works

1. Capture image from webcam or input image.
2. Detect human faces using OpenCV.
3. Crop detected face.
4. Feed the cropped image into the Age Detection model.
5. Feed the cropped image into the Gender Detection model.
6. Display predicted age and gender.
